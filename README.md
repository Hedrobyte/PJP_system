## Descrição:
- `DESCRIÇÃO DO PRODUTO:`
O sistema deverá gerenciar uma loja de vendas online de computadores, placas de vídeo e processadores, facilitando a realização de compras e vendas.

## Funcionalidades:
- `Cadastro do usuário`
- `Login do usuário`
- `Realizar compra`
- `Alterar compra`
- `Excluir compra`
- `Cadastrar produto`
- `Alterar produto`
- `Excluir produto`

## Tipos de usuários:
- `Cliente(usuário)`
- `Dono(admin)`
